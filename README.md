## Video Background

This website now uses a video background instead of a static image. 

**To add your video background:**
1. Place your video files in the root directory as:
   - `background-video.mp4` (required)
   - `background-video.webm` (optional, for better browser support)

2. See [VIDEO_BACKGROUND_INSTRUCTIONS.md](VIDEO_BACKGROUND_INSTRUCTIONS.md) for detailed instructions on:
   - Video specifications and optimization
   - Hosting options (local vs CDN)
   - Browser compatibility
   - Performance considerations

**Fallback:** If video files are not present, the site will fall back to the original static background image.

## Quick Start

1. Clone the repository
2. Add your video files (see above)
3. Open `index.html` in a browser or deploy to your hosting service
