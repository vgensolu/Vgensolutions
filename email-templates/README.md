## üìã Best Practices

### Content
- Keep subject lines under 50 characters
- Use preheader text effectively
- Include clear call-to-action
- Personalize with recipient name when possible
- Test email content for spam triggers

### Images
- Optimize images for web (< 1MB total email size)
- Use alt text for all images
- Host images on reliable CDN
- Consider users with images disabled
- Use web-safe formats (JPG, PNG, GIF)

### Design
- Maintain consistent branding
- Use hierarchy (headings, subheadings, body)
- White space is your friend
- Test dark mode appearance
- Ensure text is readable (good contrast)

### Technical
- Include plain text version
- Add unsubscribe link (required by law)
- Include physical mailing address
- Test all links before sending
- Use email validation services

## üîß Troubleshooting

### Common Issues and Solutions

**Issue**: Images not displaying
- **Solution**: Use absolute URLs, not relative paths
- Host images on public server or CDN

**Issue**: Layout breaks in Outlook
- **Solution**: Use table-based layouts, avoid modern CSS
- Test specifically in Outlook versions

**Issue**: Gmail cuts off email
- **Solution**: Keep total email size under 102KB
- Optimize images and remove unnecessary code

**Issue**: Mobile view looks wrong
- **Solution**: Test media queries
- Ensure viewport meta tag is present

**Issue**: Colors look different in various clients
- **Solution**: Use hex codes, not color names
- Test in multiple clients
